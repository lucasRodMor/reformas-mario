# Reformas Mario 🏗️

Sitio web profesional para **Reformas Mario**, expertos en reformas integrales, albañilería, fontanería y electricidad.

## 🚀 Características

- **Diseño Moderno y Responsivo**: Optimizado para todos los dispositivos.
- **Secciones de Servicios**: Detalle de trabajos realizados.
- **Galería de Proyectos**: Visualización de reformas anteriores.
- **Formulario de Contacto**: Facilidad para solicitar presupuestos.

## 🛠️ Tecnologías

Este proyecto está construido con:
- [React](https://reactjs.org/)
- [Vite](https://vitejs.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)

## 💻 Desarrollo Local

**Requisitos previos:** Node.js (v18+)

1. **Instalar dependencias:**
   ```bash
   npm install
   ```

2. **Configurar variables de entorno:**
   Crea un archivo `.env.local` basado en `.env.example` y añade tu `GEMINI_API_KEY`.

3. **Ejecutar en modo desarrollo:**
   ```bash
   npm run dev
   ```
